# TrackGit

Tracker Addon for Git Version Control System.

## Installation

Clone the repo and run following:

```
make
cp objects.x86-cc2-debug/TrackGit ~/config/non-packaged/add-ons/Tracker/
```

or you can execute `run.sh`.

## Features Added

+ [x] Init
+ [x] Status
+ [ ] Clone
+ [ ] Add Files
+ [ ] Commit
+ [ ] Push
+ [ ] Create Branch
+ [ ] Merge Branch
