1 # TrackGit
2 
3 Tracker Addon for Git Version Control System.
4 
5 TrackGit provides user with GUIs for various Git commands. The implementation was done using [LibGit2](http://libgit2.github.com/).
6 
7 TrackGit uses the Dynamic population of menu. The commit which adds this feature in Haiku is [here](https://review.haiku-os.org/#/c/haiku/+/164/).
